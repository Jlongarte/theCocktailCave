# 🍸 The Cocktail Cave — Cocktail Bar Website

A premium, minimalist web application built for **the Cocktail Cave**, a sophisticated cocktail bar concept based in the heart of Dublin. The project features a dark cinematic aesthetic, real-time data fetching, seamless multi-page transitions, and complex responsive layouts.

## ✨ Features

* **Cinematic Design:** A dark mode interface featuring *glassmorphism* UI components, fine borders, and smooth micro-interactions.
* **Dual-Behavior Layout (Blog/News):** * **Desktop:** An elegant, editorial asymmetric double-column viewer with a fixed reading pane.
    * **Mobile/Tablet:** Seamlessly mutates into a clean, smooth accordion layout powered by Framer Motion.
* **Dynamic Booking System:** Full table reservation form with state-controlled validation, multi-step transition screens, and automated home redirection.
* **Reusable Component Architecture:** Scalable React design utilizing highly custom polymorphic components (such as the scroll-driven parallax `Hero`).
* **Performance Optimization:** Configured with an automated external health-check system (*UptimeRobot*) to bypass Render’s container cold-start limitations, ensuring sub-second response times on public endpoints.

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite.js)
* **Routing:** React Router DOM (v6)
* **Animations:** Framer Motion (Scroll-linked values, layout transitions, AnimatePresence)
* **Content Parsing:** React Markdown
* **Styling:** Pure CSS3 (Custom properties, CSS Grid, Flexbox, Fluid Typography)
* **Backend Support:** Custom REST API hosted on Render
